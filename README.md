# sc-repo-trade

## Первоначальная настройка Git
Эти команды используются для инициализации проекта с помощью Git и загрузки его на репозиторий GitHub.

### 1. Инициализация репозитория Git
```bash
git init
```
Создаёт новый локальный репозиторий Git в текущей папке проекта. Git начнёт отслеживать изменения в этом каталоге.

### 2. Добавление всех файлов проекта в Git
```bash
git add .
```
Индексирует (подготавливает к коммиту) все файлы в текущем каталоге для следующего коммита.

### 3. Создание первого коммита
```bash
git commit -m "first commit"
```
Сохраняет снимок подготовленных файлов в истории репозитория с комментарием `"first commit"`.

### 4. Переименование основной ветки
```bash
git branch -M main
```
Переименовывает текущую ветку в `main`. Это стандартное имя основной ветки для GitHub.

### 5. Подключение к удалённому репозиторию GitHub
```bash
git remote add origin https://github.com/USERNAME/REPO.git
```
Добавляет удалённый репозиторий с именем `origin`. Это связывает ваш локальный проект с репозиторием на GitHub.

### 6. Загрузка проекта на GitHub
```bash
git push -u origin main
```
Выгружает локальную ветку `main` на удалённый репозиторий GitHub.
Флаг `-u` устанавливает upstream-ветку, что позволяет в дальнейшем использовать просто `git push` для отправки изменений.
